Excel VBA


Excel 录制宏 
1.选中任意一个单元格
2.打开 菜单 工具/宏/录制新宏/重命名宏名/确定
3.设置单元格字体为仿宋体，红色
4.点击停止录宏
Excel 执行宏 
1.选中需要设置的任意单元格或单元格区域（为了看出效果，单元格最好有字符）
2.打开 菜单 工具/宏/宏，显示“宏”对话框（快捷键：Alt+F8）
3.选择我们刚才录制的宏，单击“执行”







Office 2013 VOL版 
@echo off
cd %ProgramFiles%\Microsoft Office\Office15        # 进入ospp.vbs的目录即Office的安装目录 D:\Office\Office\Office15\OSPP.VBS
cscript ospp.vbs /sethst:177.18.80.150             # 设置KMS主机名。一般为IP地址。177.18.80.150是现在可用的KMS服务的IP地址
cscript ospp.vbs /act                              # 激活当前安装的Office
cscript ospp.vbs /dstatus                          # 显示当前已安装产品密钥的许可证信息
cscript ospp.vbs /remhst                           # 删除KMS主机名
pause
ospp.vbs文件 全称：Office Software Protection Platform，即Office软件保护平台脚本，是Office自身提供的激活管理工具
C:\Program Files\Microsoft Office\Office15\OSPP.VBS
