PADS Layout 具有两种类型的栅格(Grids)
即工作栅格(Working Grids)和显示栅格(Display Grid)
工作栅格 Working Grids
PADS Layout 在设计过程中使用四类工作栅格(Working Grids)：
设计栅格 Design Grid 定义布线时的栅格大小
过孔栅格 Via Grid 定义建立过孔时的栅格大小
扇出栅格 Fanout Grid 定义扇出的栅格大小
铺铜栅格 Hatch Grid 定义铺铜的栅格大小

显示栅格 Display Grid
这种点状的栅格用于设计的辅助。
你可以设置显示栅格(Display Grid)与设计栅格匹配，或者为设计栅格的数倍大小。
为了关闭显示栅格(Display Grid)，设置显示栅格(Display Grid)等于0即可。
