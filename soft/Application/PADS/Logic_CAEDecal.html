CAE Decal

CAE Decal(CAE封装)是一个二维线(2D line)符号,是纯粹的图形
> Tools(工具) / Part Editor(元件编辑器)

创建管脚封装(Pin Decal)
>文件 / 新建 / Pin Decal(管脚封装) / 确定 /
显示 :
原点标记 --------------- 原点标记有两个用途，一是连线连接的点，另一个是移动或放置管脚封装(Pin Decal)的原点
#E ---------------------- 指示管脚号码
PNAME ----------------- 指示管脚或功能的名字，如A00、D01或VCC
#0:TYP=U SWP=0 -------- 指示管脚类型(Pin Type)和门交换值(Gate Swap Values)
NETNAME --------------- 在原理图中显示时的网络名字标记


创建CAE封装
>文件 / 新建 / CAE封装 / 确定 / 封装编辑工具栏 /
显示 :
REF ------------------ 元件的参考编号(Reference Designation)，如 : U1、U2、U3...
PART-TYPE ------------ 元件类型(Part Type)
*Free Label 1 -------- 元件类型(Part Type)的第一个属性
*Free Label 2 -------- 元件类型(Part Type)的第二个属性

1.CAE封装向导工具创建
>封装编辑工具栏 / CAE封装向导 / ...

2.CAE封装手动创建
>封装编辑工具栏 / 创建2D线 / 添加端点 / ...
