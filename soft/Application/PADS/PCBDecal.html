建立PCB封装(PCB Decal)
> Tools工具 / PCB Decal Editor封装编辑器
进入封装编辑器(Decal Editor)，字符 Name 和 Type 以及一个PCB 封装(PCB Decal)原点标记将出现。
Name 字符标号(Text Label)放在这里，代表一个元件(Part)的参考编号(Reference Designation)，如 : U1、U2、U3...
Type 代表元件类型(Part Type)

> Drafting Toolbar / Add Terminal 添加端点
> Setup/Pad Stacks 焊盘形状编辑器
设置焊盘形状(Pad Shapes)和尺寸大小(Sizes)
手工调整焊盘(Pad)尺寸和钻孔(Drill)尺寸大小

Shape: Size: Layer list box
安装面、贴装面
中间层、内层
相对面、对面

钻孔Drill 选项右边的Plated 选项:
它代表此焊盘孔的金属化选项，如果没有选中，则表面孔内壁不进行金属化处理，
这样各层之间的网络将无法连接在一起，在连通性检查时将会报告错误。
