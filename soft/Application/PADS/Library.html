Library Path C:\bin\ED\PADS\MentorGraphics\9.5PADS\SDD_HOME\Libraries\PADS_Layout_Extended_Libraries 是根据 IPC-7351 标准的封装库

PADS Library由4个不同的文件组成
.ld : CAE Decal(CAE封装/原理图符号)/逻辑封装(Logic) 如与非门、或非门
.ln : Line item(图形)/2D线 如公司LOGO
.pd : PCB Decal(PCB封装) PCB封装在OrCAD中叫做PCB Footprint，在PADS中叫做Part Type
.pt : Part Type(元件) 如IC 74LS02


Parts (元件类型)
Decals (PCB封装类型)
Logic (原理图符号类型 CAE Decal)
2D Lines (图形类型)

PADS元件由元件类型(Part Type)、逻辑封装(CAE Decal)和PCB封装3部分构成

在PADS Logic或者PADS Layout中建立元件类型(Part Type)

CAE封装(CAE Decal)只可以在PADS Logic中建立

PCB 封装(PCB Decal)只可以在PADS Layout中建立
PCB 封装(PCB Decal)是元件的物理表示，即提供它的管脚图形。
PCB 封装(PCB Decal)包含元件管脚(Component pins)的各个端点(Terminals)和元件的外框(Component outline)

PADS 元件类型(Part Type)为7404的例子
元件类型(Part Type)名字:7404
CAE 封装(CAE Decal):INV
PCB 封装(PCB Decal):DIP14
电性能参数(Electrical parameters):六个逻辑门(A 到F)使用12 个管脚、一个电源管脚和一个地线管脚。
