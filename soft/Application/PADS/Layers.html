Layers Setup
> Setup / Layer Definition / Modify(修改)... / 4 or 6 or 8 or n /
1 Top
重新命名Top层为 : Primary Component Side(主元件面)
Electrical Layer Type(电气层类型) : Component(元器件)
Plane Type(平面类型) : No Plane(无平面)
Routing Direction(布线方向 : Vertical(垂直)

2 Bottom
重新命名Bottom层为 : Secondary Component Side(次元件面)
Electrical Layer Type(电气层类型) : Component(元器件)
Plane Type(平面类型) : No Plane(无平面)
Routing Direction(布线方向) : Horizontal(水平)

4层板之 :
Inner Layer 2
重新命名为 : Ground Plane(地平面层)
Plane Type(平面类型) : CAM Plane
Assign Nets(分配网络) / Select GND / ok /
Routing Direction(布线方向) : Any(任意)

Inner Layer 3
重新命名为 : Power Plane(电源平面层)
Plane Type(平面类型) : Split/Mixed(分割/混合)
Assign Nets(分配网络) / Select 5V and 12V / ok /
Routing Direction(布线方向) : Any(任意)

设置层的Stackup(叠层)
Thickness(厚度)
设置层的厚度和叠层(Stackup)参数
Copper Thickness Units(铜厚单位)设置为 : Design(mils)
不要设置为 : 重量(盎司)
