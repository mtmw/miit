Windows记事本中的编码 ANSI、Unicode、Unicode big endian、UTF-8

big endian和little endian是CPU处理多字节数的不同方式
例如“a”的Unicode编码是0061,那么写到文件里时，究竟是将00写在前面，还是将61写在前面？
如果将00写在前面，就是big endian,如果将61写在前面，就是little endian。
endian”这个词出自《格列佛游记》。小人国的内战就源于吃鸡蛋时是究竟从大头(Big-Endian)敲开还是从小头(Little-Endian)敲开，
一般将endian翻译成“字节序”，将big endian和little endian称作“大尾”和“小尾”。

Unicode编码前面都有几个标记字符，这些字符放在文本文件的开头，称为“ BOM（ Byte Order Mark，字节顺序标记）”指明了文本的编码方式 
编码                     BOM 值
UTF-8(无BOM)             无

UTF-8                    ef bb bf                       #  Microsoft的记事本在保存为 UTF-8 编码时，会在每个文件开头添加三个字节 ef bb bf 的字符

UTF-16 big endian        fe ff                          #  00在前面

UTF-16 little endian     ff fe                          #  00在后面

UTF-32 big endian        00 00 FE FF

UTF-32 little endian     FF FE 00 00


Unicode 学名是"Universal Multiple-Octet Coded Character Set"，简称为UCS。UCS可以看作是"Unicode Character Set"的缩写
UCS规定了怎么用多个字节表示各种文字。怎样传输这些编码，是由UTF(UCS Transformation Format)规范规定的，常见的UTF规范包括UTF-8、UTF-7、UTF-16。
UCS有两种格式：UCS-2和UCS-4。顾名思义，UCS-2就是用两个字节编码，UCS-4就是用4个字节（实际上只用了31位，最高位必须为0）编码
UTF = UCS Transformation Format = UCS转换格式 它是将Unicode编码规则和计算机的实际编码对应起来的一个规则。现在流行的UTF有2种：UTF-8和UTF-16。

Storage
硬盘制造商是采用十进制给GB下定义1GB = 1,000,000,000字节，而计算机科学传统上是采用二进制即1GB = 1,073,741,824（1024^3）比特
Windows和linux操作系统中采用的是后一种定义 MAC OS X采用前一种定义

吉字节（GB、Gigabyte），常简写为G
1GB = 1,000 MB = 103 MB
1GB = 1,000,000 KB = 106 KB
1GB = 1,000,000,000 B = 109 B

32Gb(4096M x 8bit) = 4GB

不过，Gigabyte往往可以指Gibibyte，其换算是：
1GiB = 1,024 MiB = 210 MiB
1GiB = 1,048,576 KiB = 220 MiB
1GiB = 1,073,741,824 B = 230 MiB

兆字节（MB、Megabyte），常简写为B
1MB = 1,000 KB
1MB = 1,000,000 B
1GB = 1,000 MB
1TB = 1,000,000 MB

另外，Megabyte往往指Mebibyte，其换算是：
1MiB = 1,024 KiB
1MiB = 1,048,576 B = 10242 B
1GiB = 1,024 MiB = 1,048,576 KiB = 10242 KiB
1TiB = 1,024 GiB = 1,073,741,824 KiB = 10243 KiB 



















宽带业务有1M、2M、4M、6M、8M、10M、20M、50M、100M等接入,其速率单位为bps(比特/秒)


运营商向客户提供宽带业务时，按上行速率最高可达（upload speed up to）和下行速率最高可达（download speed up to）进行承诺。


上网带宽包括上行速率（upload）和下行速率（download），上行指客户发送信息，下行指客户接收信息。


1Byte/s=8bps


1KB/sec=8Kbps = 8192bps


1Kbps=1024bps


1M套餐宽带理论速率为: 1Mbps = 1024Kbps = 128KB/s


2M套餐宽带理论速率为: 2Mb/s = 2048Kb/s = 256KB/s


4M（即4Mb/s=4096Kb/s）宽带理论速率是：512KB/s


 1Mbps=1024Kbps 例如：2Mbps=2048Kbps=256KB/sec


关于bit(比特)/second(秒)与Byte(字节)/s(秒)的换算说明:线路单位是bps,表示 bit(比特)/second(秒),注意是小写字母b.


用户在网上下载时显示的速率单位往往是KB/s Byte(字节)/s(秒)，注意是大写字母B。





位 bit(比特)是表示信息的最小单位,叫做二进制位,一般用0和1表示


Byte叫做字节,由8个位（8bit）组成一个字节(1Byte),用于表示计算机中的一个字符


bit与Byte之间可以进行换算，其换算关系为：1Byte=8bit（或简写为：1B=8b）


在实际应用中一般用简称,即1bit简写为1b(注意是小写英文字母b),1Byte简写为1B（注意是大写英文字母B）
