Server IP    # your hostname or server IP (IPv4/IPv6).
Server Port  # server port number.
Password
SOCKS 5 Proxy Port  # 本地端口 默认为 1080
Encryption Method  # AES-256-CFB 默认的加密形式
Timeout in Second  # 600 默认的超时 一般不用改

字段名 含义
Server IP 服务端监听地址(IPv4或IPv6)，服务端可填写 0.0.0.0
Server Port 服务端的端口
local_address 本地监听地址，缺省为127.0.0.1
local_port 本地端的端口
password 用于加密的密码
timeout 超时时间，单位秒
method 加密方法，推荐 "aes-256-cfb"
fast_open 是否使用 TCP_FASTOPEN, true / false
workers worker 数量，Unix/Linux 可用，如果不理解含义请不要改

白、黑名单
菜单栏 > Profile配置文件 >
> Proxification Rules...代理规则 >
> Add... > Name: Firefox > Applications firefox.exe > Action动作: Proxy SOCKS5 127.0.0.1 || Direct直接(即不通过代理) || Block拦截 >

开始默认是有2条策略，就是 Default 和 Localhost，Default是不能被删除的
如果要使计算机全局使用代理上网，只需要将Default修改为想要通过的代理即可
默认为”Direct”，即不通过代理访问

例: 将虚拟路由热点 VirtualRouter.exe 通过代理连接Shadowsocks
1. 将 Default 的Action动作设置为 Direct直接(即不通过代理)
2. Proxification Rules...代理规则 > Add... > Name: VirtualRouter > Applications VirtualRouter.exe > Action动作: Proxy SOCKS5 127.0.0.1 >
