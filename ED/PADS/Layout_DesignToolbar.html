布线(Route) :
> 设计工具栏(Design Toolbar) / 添加布线(Add Route(F2))
Ctrl+Alt+S 打开状态栏
动态布线(Dynamic Route(F3))、草图布线(Sketch Route)、自动布线(Auto Route(F7))、总线布线(Bus Route(Ctrl+Alt+B))
开始布线模式下,只需移动光标至合适位置,再点击左鼠键或按空格键确定一段段的走线
在布线时改变布线角度模式,右键菜单"角度模式(Angle Mode)"
在布线时当需要停下时可按Ctrl+左鼠点击或使用右键菜单,点击"结束(End(Ctrl+LButton+Click))"以停下走线
在布线时按退格键将依次取消最后一段走线
在布线时使用右键菜单的"交换结束点(Swap End)"切换至网络的另一端进行布线

在布线时变换线宽 cmd wn n为线宽数字
在布线时添加圆弧走线段,右键"添加圆弧(Add Arc)".圆弧大小,可通过移动光标调整

移动线段
选择某一段布线 / 右键 / 移动(Move (Ctrl+E))
or
选择某一段布线 / 左键直接移动

拉伸线段
选择某一段布线 / 右键 / 拉伸(Stretch (Shift+S))
or
选择某一段布线 / 左键直接拉伸

直角线段转换为圆弧
点选直角 / 右键 / 转换为圆弧(Convert To Arc) / 移动光标,调整圆弧大小(此时按住或不按住左键都可以)

添加回路布线(即已经布好的线再从旁边增加布线,形成一个回路)
选择某一段布线 / 右键 / 回路布线(Route Loop (Ctrl+J))

在线段添加拐角
选择某一段布线 / 右键 / 添加拐角(Add Corner) / 移动光标至合适位置,左鼠点击完成添加拐角

分割线段
选择某一段布线 / 右键 / 分割(Split) / 移动光标至合适位置,左鼠点击完成分割线段


倒角设置
Tools / Options / 设计(Design) / 倒角(Miters) /
/ 斜交(Diagonal) or 圆弧(Arc) /
自动倒角(Auto miters)
点选直角 / 右键 / 添加倒角(Add Miters) / 直角变成了倒角 or 弧形倒角
整根线添加倒角
选择某一段布线 / 右键 / 选择管脚对(Select Pin Pair (F5)) / 在高亮的布线上右键 / 添加倒角(Add Miters) / 则整根线上的直角都倒角了
用类似的方法可以删除倒角(Delete Miters)

将线段换层
选择某一段布线 / 右键 / 特征(Properties(Alt+Enter)) / 层(Layer) 下拉选择相应的层(线段换层后会自动添加过孔) /

保护个别导线
选择某一段布线 / 右键 / 选择管脚对(Select Pin Pair (F5)) / 在高亮的布线上右键 / 特征(Properties(Alt+Enter)) / 勾选 保护布线(Protect Routes)
如需要显示区分保护和没有保护的导线
Tools / Options / 布线(Routing) / 常规(General) / 勾选 显示保护(Show protection) /
这样设置为保护的导线显示便会非实心,设置为保护的线,便不能被修改或删除了


在布线时可通过 cmd : ln 自动添加过孔并切换到第n层,n为层数字
在布线时可通过右键菜单"层切换(Layer Toggle(F4))"切换层,注意这个不像cmd ln 可随意切换至任何层,
而是按层对(Layer Pair)设定作切换的
工具/选项/布线(Routing)/常规General)/层对(Layer Pair)
层对(Layer Pair) cmd : pl 1 3 设定首个(First)为第一层及(Second)为第三层


>绘图工具盒(Drafting Toolbar) / 平面区域(Plane Area)

在布线时设置 : 右键 / 以过孔结束模式(End Via Mode) /
or 以没有过孔结束(End No Via)
or 以过孔结束(End Via)
or 以测试点结束(End Test Point)
则当"Ctrl+左键"暂停走线时会对应执行上面的选择
结束过孔模式的cmd : e 是单向循环切换



在布线时添加过孔 cmd : v、vt 就会弹出过孔选择，你可以选择你已经设置好的过孔，点OK后
你的光标上就附有你刚选择的那个过孔类型了。不过，一般同一个网络不会增加不同类型的过孔，
例如电源线一般用大孔，信号线一般用小孔，所以我是在规则设置里把不同的网络设置为自己想要的过孔，
这样在走线时或另加过孔时就会自动用到你需要的过孔了。

n后面加网络名高亮网络,再输入n取消高亮
