设计(Design)

有四种在线设计规则检查(On-line Design Rule Checking (DRC))方式 :
DRC Prevent(DRC防止) : 在布局、布线和布线修改期间，禁止建立冲突
cmd : DRP

DRC Warn(DRC告警) : 在布局和布线期间，生成出错信息报告 但是允许你继续建立空间冲突(Violations)。 在DRC Warning方式下的布线和布线修改禁止建立冲突或导线插入，就象DRC Prevent 一样
cmd : DRW

DRC Ign Clr(忽略DRC) : 在布线期间，防止导线的插入，但是其它的工作同 DRC Off 一样。 你可以从DRC Ignore Clearance 状态快速地切换到DRC Prevent 状态，它比从DRC Off 状态切换到DRC Prevent状态要快一点，因为DRC Ignore Clearance 是DRCPrevent 的一个子集，而DRC Off 不是。
cmd : DRI

DRC Off(关闭DRC) : 指明不进行检查
在布局和布线期间的 规则的冲突是允许的。
安全间距冲突和插入导线是不禁止的。
cmd : DRO


栅格和捕获/栅格(Grids And Snap/Grids)

PADS Layout 具有两种类型的栅格(Grids)
即工作栅格(Working Grids)和显示栅格(Display Grid)
工作栅格 Working Grids
PADS Layout 在设计过程中使用四类工作栅格(Working Grids)：
设计栅格 Design Grid 定义布线时的栅格大小
过孔栅格 Via Grid 定义建立过孔时的栅格大小
扇出栅格 Fanout Grid 定义扇出的栅格大小
铺铜栅格 Hatch Grid 定义铺铜的栅格大小

显示栅格 Display Grid
这种点状的栅格用于设计的辅助。
你可以设置显示栅格(Display Grid)与设计栅格匹配，或者为设计栅格的数倍大小。
为了关闭显示栅格(Display Grid)，设置显示栅格(Display Grid)等于0即可。
