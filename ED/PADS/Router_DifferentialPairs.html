差分对(Differential Pairs)

PADS Router 与 PADS Logic、PADS Layout 进行交互
在 PADS Router 中选中差分对网络 / 右键 / 建立差分网络(Make Differential Net)
在以下路径查看已建立的差分对网络
项目浏览器(Project Explorer) / 网络对象(Net Objects) / 差分对(Differential Pairs)
选中差分对 / 右键 / 特性(Properties) / 设置如下特性阻抗

特性阻抗 :
差分 100Ω :
表层 所有层(All layers) 宽度(Width)4.4mils 间隙(Gap)8.5mils
内层 选择内层中的走线层 宽度(Width)4.6mils 间隙(Gap)8mils

差分 90Ω :
表层 所有层(All layers) 宽度(Width)5mils 间隙(Gap)7mil
内层 选择内层中的走线层 宽度(Width)5.5mils 间隙(Gap)8mils

勾选 允许自动布线时穿越障碍
