Cadence Allegro16.6安装和破解步骤:
install:
安装License Manager
1.双击bin\setup.exe，弹出安装界面
2.点击第一项 License Manager
3.点击"Next"
4.选择第一项I accept the terms of the license agreement
点击"Next" 5.出现菜单Cadence License Manager 12.01
默认License Manager路径为C:\Cadence\LicenseManager
更改License Manager路径为C:\ED\Cadence\LicenseManager 点击"Next"
6.出现菜单Cadence License Manager 12.01
点击"Next"， 8.点击Install
弹出菜单Cadence License Server Configuration - Version 1.0.20) 会运行进度条
弹出Cadence License Server Configuration 对话框
点击 Cancel
点击"是" 11.点击"Finish"，完成License Manager的安装

12.接下来安装Cadence的第二项即Product installation
点击Product installation
13.点击"Next"
14.选择第一项I accept the terms of the license agreement
点击"Next" 15.出现菜单Cadence SPB/OrCAD 16.6
选择第一个Complete(全部安装)。 选择下面第二个Anyone who uses this computer all users所有用户
点击"Next" 18.选择路径，
默认安装路径C:\Cadence\SPB_16.6更改为C:\bin\ED\Cadence\SPB_16.6 默认工作路径C:\SPB_Data更改为D:\ED\Cadence\SPB_16.6_Data
点击"Next" 19.点击Install，需要安装十多分钟。
20.点击"Finish"，还等二十多秒待界面消失。
21.安装cadence16.6最新补丁bin\Hotfix_SPB16.60.047_wint_1of1.直到finish
22.若是64位系统，复制bin\Cadence16.6_crack\cdslmd.exe到LicenseManager的安装目录C:\bin\ED\Cadence\LicenseManager,并替换原文件

crack:
1.在任务管理器中确认一下是否有这两个进程,有就结束掉,即cdsNameServer.exe和cdsMsgServer.exe,一般没有。

2.把安装目录下的C:\bin\ED\Cadence\SPB_16.6\tools\pspice目录下的
orSimSetup.dll剪切出来找个地方先放着不理
（然后破解完毕再拷回原处,如果不用仿真部分删掉也无所谓）。
3.把bin\Cadence16.6_crack\crack\LicenseManager目录中的
pubkey、pubkey.exe和LicenseManager.bat 3个文件放到
C:\bin\ED\Cadence\LicenseManager目录下并运行LicenseManager.bat。
4.把bin\Cadence16.6_crack\crack\SPB_16.5\tools目录中的
pubkey、pubkey.exe和Tools.bat 3个文件放到
C:\bin\ED\Cadence\SPB_16.6\tools目录下并运行Tools.bat
（注意看一下DOS窗口运行差不多一分钟就说明破解成功）
5.删除bin\Cadence16.6_crack\crack\LicGen里的license.lic,
然后双击licGen.bat生成新的license.lic 把这新的license.lic文件拷到C:\bin\ED\Cadence\LicenseManager目录下
7.计算机开始菜单/所有程序/Cadence/License Manager/License Servers Configuration Unilily,
弹出的对话框中点browes...指向刚才生成的C:\usr\ID\Cadence\LicenseManager\license.lic
点击"Next"后弹出菜单，将Host Name主机名改成改里边的this-host为自己电脑的名字 mPad
点击"Next"
点击Finish
11.计算机开始菜单/所有程序/Cadence/License Manager/License Client Configuration Unility，
填入5280@(你的主机名),点下一步(next),最后点finish,直到完成.
点下一步(next),最后点 finish
12.控制面板/系统/高级系统设置/高级/环境变量，系统变量和环境变量里面找到CDS_LIC_FILE ，
编辑内容为C:\bin\ED\Cadence\LicenseManager\license.dat。到此,破解完成. 不必重启电脑就可运行程序。
说明：指向内容为dat文件而不是lic文件。 13.将第2步剪切出来的文件归位
